# Maze Game 🌀

A fun little maze game made with Python and Turtle Graphics.

## 🎮 How to Play
- Use the arrow keys to move your player.
- Collect all the sparkles ✨.
- Avoid the enemies!
- When you collect all sparkles, the game ends.
